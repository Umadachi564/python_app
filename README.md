# 動作バージョン情報
- Python: 3.8.0 以上
- Pytorch: 1.7.0 以上

# 挙動確認
- ローカルPCで構築したDockerコンテナ環境下で正常に作動. 
- Render.comの無料枠だとRAMが少なすぎてwebアプリが動きませんでした. (悲しいです) (2022年10月13日)
